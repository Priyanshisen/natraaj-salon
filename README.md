# Natraaj Beauty Salon 🌸

Welcome to the official website repository for **Natraaj Beauty Salon**, a premium ladies beauty parlour run by Nirmala Sen with over 15 years of expertise. 

The salon offers expert hair, skin, and bridal services, along with handcrafted, 100% natural herbal products.

## ✨ Features
- **Responsive Design:** Beautifully designed UI that works seamlessly across desktop, tablet, and mobile devices.
- **Premium Aesthetics:** Feminine, elegant theme with glassmorphism effects, floating petal animations, and soft color palettes.
- **Custom Order System:** Integrated email-based order system for purchasing homemade herbal products (Hair Henna & Hair Oil).
- **Appointment Booking:** Secure and easy-to-use booking system for scheduling salon services.

## 🛠️ Tech Stack
- **HTML5:** Semantic and accessible markup.
- **CSS3:** Custom styling, CSS variables, flexbox/grid layouts, and keyframe animations. No external frameworks used for maximum performance.
- **Vanilla JavaScript:** DOM manipulation, scroll reveal animations, and modal logic.
- **FormSubmit:** Secure email forwarding for appointments and orders.

## 🚀 Deployment
This website is a static single-page application and is optimized to be deployed seamlessly on **Vercel**, Netlify, or GitHub Pages.
